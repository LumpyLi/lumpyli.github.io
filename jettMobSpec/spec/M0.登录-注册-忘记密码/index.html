    Spec¶

    Note

    M0-1. 登录 or 注册页面¶

    1. 点击 登录 按钮进入 M0-2. 登录首页
    2. 点击 注册 按钮进入 M0-3. 注册首页

    M0-2. 登录首页¶

    1. 点击 返回 按钮返回至 M0-1. 登录 or 注册页面
    2. 点击 手机号码 弹出 数字键盘，见 M0-2-1. 登录 - 输入手机号码
    3. 点击 密码 弹出 普通键盘 进行输入密码，见 M0-2-2. 登录 - 输入登录密码
    4. 点击 切换可见 按钮切换密码可视状态，默认 不可视
    5. 手机号及密码输入完毕后可点击 登录 按钮进入 M0-2-3. 登录 - 登录中
    6. 点击 注册 按钮进入 M0-3. 注册首页
    7. 点击 忘记密码 按钮进入 M0-4. 忘记密码

    M0-2-1. 登录 - 输入手机号码¶

    1. 点击 返回 按钮返回至 M0-1. 登录 or 注册页面
    2. 光标定位在手机号码处，弹出数字键盘
      • 从手机号码输入处失焦后检测到手机没有网络，则进入 M0-5-1. 全局网络不可用
      • 从手机号码输入处失焦后检测到帐号未注册，则进入 M0-5-2. 手机号未注册
      • 从手机号码输入出失焦后检测到帐号不是 11 位，则进入 M0-5-3. 手机号格式不正确
    3. 点击 密码 弹出 普通键盘 进行输入密码，见 M0-2-2. 登录 - 输入登录密码
    4. 点击 切换可见 按钮切换密码可视状态，默认 不可视
    5. 手机号及密码输入完毕后可点击 登录 按钮进入 M0-2-3. 登录 - 登录中
    6. 点击 注册 按钮进入 M0-3. 注册首页
    7. 点击 忘记密码 按钮进入 M0-4. 忘记密码
    8. 点击屏幕空白处收起数字键盘

    M0-2-2. 登录 - 输入登录密码¶

    1. 点击 返回 按钮返回至 M0-1. 登录 or 注册页面
    2. 点击 手机号码 弹出 数字键盘，见 M0-2-1. 登录 - 输入手机号码
    3. 光标定位在密码输入处，弹出样式为 完成 的键盘
    4. 点击 清除 按钮清空输入框内的内容
    5. 点击 切换可见 按钮切换密码可视状态，默认 不可视
    6. 手机号及密码输入完毕后可点击 登录 按钮进入 M0-2-3. 登录 - 登录中
    7. 点击 注册 按钮进入 M0-3. 注册首页
    8. 点击 忘记密码 按钮进入 M0-4. 忘记密码
    9. 点击屏幕空白处收起数字键盘

    M0-2-3. 登录 - 登录中¶

    1. 分以下五种情况：
      • 若网络正常并检测到已绑定对应设备，根据最新绑定的设备分别进入 M1-1. 小乐迪设备主页 或 M1-2. 大乐迪设备主页
      • 若网络正常但绑定任何设备，则进入 M1-1. 小乐迪设备主页
      • 若网络不可用则进入 M0-5-1. 全局网络不可用
      • 若网络正常并检测到登录密码不正确，则进入 M0-5-4. 帐号密码不正确
      • 若网络正常并检测到绑定时没有填写完整宝贝信息，则进入 M6-7. 添加宝贝

    M0-3. 注册首页¶

    1. 点击 返回 按钮返回前一个页面，区分 M0-1. 登录 or 注册页面 和 M0-2. 登录主页
    2. 点击进行输入手机号码，见 M0-3-1. 注册 - 输入手机号码
    3. 当输入未注册过的手机号后 获取验证码 按钮变为可点击状态，见 M0-3-2. 注册 - 获取验证码
    4. 点击输入验证码，见 M0-3-3. 注册 - 输入验证码
    5. 点击进行输入密码，见 M0-3-4. 注册 - 输入注册密码
    6. 当以上信息均填写完毕后， 注册 按钮变为可点击状态，点击后进入 M0-3-5. 注册 - 注册中
    7. 点击 用户协议 按钮进入 M0-3-6. 注册 - 用户协议

    M0-3-1. 注册 - 输入手机号码¶

    1. 点击 返回 按钮返回前一个页面，区分 M0-1. 登录 or 注册页面 和 M0-2. 登录主页
    2. 光标定位在手机号码处，弹出数字键盘，此处仅能输入数字
      • 从手机号码输入处失焦后检测到手机没有网络，则进入 M0-5-1. 全局网络不可用
      • 从手机号码输入处失焦后检测到帐号已注册，则进入 M0-5-5. 手机号已注册
      • 从手机号码输入出失焦后检测到帐号不是 11 位，则进入 M0-5-6. 手机号格式不正确
    3. 当输入未注册过的手机号后 获取验证码 按钮变为可点击状态，见 M0-3-2. 注册 - 获取验证码
    4. 点击输入验证码，见 M0-3-3. 注册 - 输入验证码
    5. 点击进行输入密码，见 M0-3-4. 注册 - 输入注册密码
    6. 当以上信息均填写完毕后， 注册 按钮变为可点击状态，点击后进入 M0-3-5. 注册 - 注册中
    7. 点击屏幕空白处收起键盘

    M0-3-2. 注册 - 获取验证码¶

    1. 点击 返回 按钮返回前一个页面，区分 M0-1. 登录 or 注册页面 和 M0-2. 登录主页
    2. 手机号码输入出失焦后检测没注册过，获取验证码 按钮变为可点击状态
    3. 点击获取验证码，系统发送一条验证码信息给用户手机，并有对应提示，见 M0-3-2-1. 注册 - 验证码发送提示。获取验证码 按钮在倒计时 60 秒结束后可重新点击
      • 若短信平台返回 获取验证码超出每天次数 的错误信息，则进入 M0-5-7. 验证码超出每天次数
      • 若短信平台返回 该手机号已被拉入黑名单 的错误信息，则进入 M0-5-8. 手机号码被拉入黑名单
    4. 点击输入验证码，见 M0-3-3. 注册 - 输入验证码
    5. 点击进行输入密码，见 M0-3-4. 注册 - 输入注册密码
    6. 当以上信息均填写完毕后， 注册 按钮变为可点击状态，点击后进入 M0-3-5. 注册 - 注册中
    7. 点击 用户协议 按钮进入 M0-3-6. 注册 - 用户协议

    M0-3-2-1. 注册 - 验证码发送提示¶

    Success

    此提示为通知类 toast 提示

    1. 提示停留 3 秒之后收起
    2. 点击 关闭 按钮可以立即关闭提示，返回 M0-3-2. 注册 - 获取验证码
    3. 点击进行输入手机号码，见 M0-3-1. 注册 - 输入手机号码
    4. 点击进行输入密码，见 M0-3-4. 注册 - 输入注册密码
    5. 获取验证码 按钮在倒计时 60 秒结束后可重新点击
    6. 点击进行输入密码，见 M0-3-4. 注册 - 输入注册密码
    7. 当以上信息均填写完毕后， 注册 按钮变为可点击状态，点击后进入 M0-3-5. 注册 - 注册中
    8. 点击屏幕空白位置收起键盘

    M0-3-3. 注册 - 输入验证码¶

    1. 点击 返回 按钮返回前一个页面，区分 M0-1. 登录 or 注册页面 和 M0-2. 登录主页
    2. 点击进行输入手机号码，见 M0-3-1. 注册 - 输入手机号码
    3. 点击获取验证码，见 M0-3-2. 注册 - 获取验证码
    4. 光标定位在验证码输入处，弹出数字键盘
    5. 点击进行输入密码，见 M0-3-4. 注册 - 输入注册密码
    6. 当以上信息均填写完毕后， 注册 按钮变为可点击状态，点击后进入 M0-3-5. 注册 - 注册中
    7. 点击屏幕空白处收起键盘

    M0-3-4. 注册 - 输入注册密码¶

    1. 点击 返回 按钮返回前一个页面，区分 M0-1. 登录 or 注册页面 和 M0-2. 登录主页
    2. 点击进行输入手机号码，见 M0-3-1. 注册 - 输入手机号码
    3. 点击获取验证码，见 M0-3-2. 注册 - 获取验证码
    4. 点击输入验证码，见 M0-3-3. 注册 - 输入验证码
    5. 光标定位在密码输入处，弹出样式为 完成 的键盘（仅允许输入数字、英文及常见字符）。点击 切换可见 按钮切换密码可视状态，默认 不可视。密码允许输入数字、英文及常见字符（如 !@#$%^&*()_+ 等，但不包括空格）
      • 密码输入框可输入 6 - 32 个字符，当输入超过 32 个字符时，点击 注册 按钮弹出 M0-3-4-1. 注册 - 输入注册密码 - 密码不符合要求
      • 密码若含有空格，点击 注册 按钮弹出 M0-3-4-2. 注册 - 输入注册密码 - 密码含有空格
    6. 当以上信息均填写完毕且合乎要求后， 注册 按钮变为可点击状态，点击后进入 M0-3-5. 注册 - 注册中
    7. 点击屏幕空白处收起键盘
    8. 点击 完成 按钮同 注册 按钮进入 M0-3-5. 注册 - 注册中

    M0-3-4-1. 注册 - 输入注册密码 - 密码不符合要求¶

    Failure

    此提示为警告 / 错误类 toast 提示

    1. 提示停留 3 秒之后收起
    2. 点击 关闭 按钮可以立即关闭提示，返回 M0-3-4. 注册 - 输入注册密码

    M0-4. 忘记密码¶

    Note

    此处的交互与前面的一样，不再详述

    1. 点击 返回 按钮返回 M0-2. 登录首页
    2. 点击进行输入手机号码
      • 手机号码输入出失焦后检测已注册过，获取验证码 按钮变为可点击状态
      • 手机号码输入出失焦后检测没注册过，则弹出提示，见 M0-4-1. 忘记密码 - 手机号码未注册
    3. 手机号码输入出失焦后检测已注册过，获取验证码 按钮变为可点击状态，点击可获取验证码
    4. 点击输入验证码
    5. 点击输入新密码
    6. 当以上信息均填写完毕后，登录 按钮变为可点击状态，点击后进入 M0-2-3. 登录 - 登录中
