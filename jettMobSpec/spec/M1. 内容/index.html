    M1-1-2. 内容主页 - Default¶

    1. 点击 设置 按钮弹出点播控制面板，见 M1-1-3. 设置动作 
      • 当此帐号未绑定乐迪时，设置 按钮不显示
    2. 点击 搜索 进入 M1-1-4. 搜索 ，以热词（根据运营变化）作为 Placeholder
    3. 分类入口，点击其中一个 分类图标 进入 M1-2. 分类列表 
    4. 推荐列表名
    5. 点击 查看全部 按钮进入 M1-3. 专辑类别列表 
    6. 推荐列表里面的专辑，包含 专辑封面 及 专辑名称，专辑名称 超过两行的时候加 ...。点击进入 M1-4. 专辑页面

    Note

    滑动 M1-1-2. 内容主页 - Default 后的 Navigation Bar 的变化

    M1-1-2-2. 全局 - 加载失败 / 手机没网络¶

    Failure

    由于手机网络不好导致内容页加载失败

    1. 点击 重新加载 按钮进入 M1-1-2-3. 全局 - 重新加载 

    M1-1-2-3. 全局 - 重新加载¶

    1. 尝试重新加载内容页面
      • 加载成功：进入 M1-1-2. 内容主页 - Default 
      • 加载失败：返回 M1-1-2-2. 全局 - 加载失败 / 手机没网络 

    M1-1-3. 设置动作¶

    1. 点击 关闭 按钮关闭窗口，返回至 M1-1-2. 内容主页 - Default 
    2. 点击 Switch 开关可以控制点播时乐迪是否跳舞，默认**关闭**。即默认状态下，点播歌曲小乐迪不会跳舞（定制歌曲不受影响，如《青春修炼手册》）
    3. 点击 Overlay 关闭窗口，返回至 M1-1-2. 内容主页 - Default 

    M1-1-4. 搜索¶

    1. 显示热词，光标定位在搜索框
    2. 点击 取消 按钮返回至 M1-1-2. 内容主页 - Default 
    3. 显示不同的 热词，共 5 个，根据运营调整变化
    4. 在搜索框未输入任何 Keyword 的时候，点击 搜索 按钮直接以搜索框的 Placeholder 作为关键字搜索。即此时以 超级飞侠 作为关键字搜索。搜索结果根据返回数据有不同显示：
      • 有对应关键字结果： M1-1-4-2. 搜索 - 有结果 
      • 没有对应关键字结果： M1-1-4-3. 搜索 - 没结果 

    M1-1-4-1. 搜索 - Keyword¶

    1. 输入 Keyword 后，搜索框出现 Clear Button，点击可清空输入框内容，返回至 M1-1-4. 搜索 
    2. 点击 取消 按钮返回至 M1-1-2. 内容主页 - Default 
    3. 显示不同的 热词，共 5 个，根据运营调整变化
    4. 点击 搜索 按钮以 Keyword 作为关键字搜索。即此时以 阿凡提与猪肉干 作为关键字搜索。搜索结果根据返回数据有不同显示：
      • 有对应关键字结果： M1-1-4-2. 搜索 - 有结果 
      • 没有对应关键字结果： M1-1-4-3. 搜索 - 没结果 

    M1-1-4-2. 搜索 - 有结果¶

    1. 点击 Clear Button 可清空输入框内容，返回至 M1-1-4. 搜索 
    2. 点击 取消 按钮返回至 M1-1-2. 内容主页 - Default 
    3. 点击随意一个结果直接进行点播操作，并在内容页面内全局出现 ** 播放控制面板**（具体交互见 M1-5-1. 点播中 ）。搜索结果以音频形式展现，包含：
      • 专辑封面
      • 音频名称
      • 音频时长

    M1-1-4-3. 搜索 - 没结果¶

    1. 点击 Clear Button 可清空输入框内容，返回至 M1-1-4. 搜索 
    2. 点击 取消 按钮返回至 M1-1-2. 内容主页 - Default 
    3. 显示无相关内容
    文案¶
    无相关内容
没找到与“Keyword”相关的内容，或者试试其他的关键词吧

    M1-2. 分类列表¶

    1. 点击 返回 按钮返回至 M1-1-2. 内容主页 - Default 
    2. 显示分类列表专辑，包含 专辑封面 及 专辑名称，专辑名称 超过两行的时候加 ...。点击进入 M1-4. 专辑页面

    M1-3. 专辑类别列表¶

    1. 点击 返回 按钮返回至 M1-1-2. 内容主页 - Default 
    2. 显示推荐列表专辑，包含 专辑封面 及 专辑名称，专辑名称 超过两行的时候加 ...。点击进入 M1-4. 专辑页面

    M1-4. 专辑页面¶

    1. 点击 返回 按钮返回至前一个页面（根据来源页）
    2. 显示 专辑封面 及专辑名称
    3. 点击 播放全部 按钮按列表顺序，从第一首开始顺序循环点播。右侧显示此专辑合计的 音频数目。点击时进入 M1-5. 发送点播指令中 状态
    4. 显示 单曲名称 及 音频时长。点击该音频时，从该音频开始顺序循环点播。点击时进入 M1-5. 发送点播指令中 状态

    M1-4-1. 专辑页面 - 滑动¶

    Note

    滑动 M1-4. 专辑页面

    M1-5. 发送点播指令中¶

    1. 发送点播指令中：
      • 若点播成功，则进入 M1-5-1. 点播中 
      • 若由于乐迪没网络造成点播失败，则进入 M1-5-2. 发送点播指令失败 - 乐迪没网络 
      • 若由于手机没网络造成点播失败，则进入 M1-5-3. 发送点播指令失败 - 手机没网络 
      • 若由于没绑定乐迪造成点播失败，则进入 M1-5-4. 发送点播指令失败 - 尚未绑定乐迪 

    M1-5-1. 点播中¶

    1. 显示 专辑封面、音频名称 及 专辑名称，点击此区域进入到该专辑的页面，即 M1-4. 专辑页面 ，且将该曲目定位到屏幕中央
    2. 点击 下一曲 按钮切歌，播放下一曲
    3. 点击 展开 按钮展开播放控制面板，见 M1-6. 播放器 

      播放控制面板在内容页面内全局出现

    M1-5-2. 发送点播指令失败 - 乐迪没网络¶

    Failure

    此提示为警告 / 错误类 toast 提示

    1. 提示停留 3 秒之后收起
    2. 点击 关闭 按钮可以立即关闭提示，返回至 M1-4. 专辑页面 

    M1-5-3. 发送点播指令失败 - 手机没网络¶

    1. 点击 我知道了 按钮关闭 Alert，返回至 M1-4. 专辑页面 

    M1-5-4. 发送点播指令失败 - 尚未绑定乐迪¶

    1. 点击 去绑定 按钮进入 M2-2. 切换机器人 后再走绑定流程
    2. 点击 取消 按钮关闭 Alert，返回至 M1-4. 专辑页面 

    M1-6. 播放器¶

    1. 显示 专辑封面、音频名称 及 专辑名称，点击此区域进入到该专辑的页面，即 M1-4. 专辑页面 ，且将该曲目定位到屏幕中央
    2. 点击 收起 按钮收起播放控制面板，见 M1-5-1. 点播中 
    3. 控制上一曲、停止、下一曲。其中点击 停止 按钮，播放控制面板隐藏，并停止对 RBT 进行点播操作，RBT 返回至 待机 状态
    4. 控制 RBT 音量加减

      播放控制面板在内容页面内全局出现
