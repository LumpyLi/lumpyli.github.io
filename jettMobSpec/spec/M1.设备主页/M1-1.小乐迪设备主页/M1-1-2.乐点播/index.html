    M1-1-2-0. 资源架构¶

    此页面用于列举当前乐点播的资源架构：

    1. 资源分类

      • 每个分类至少包含 1 张专辑资源
      • 分类支持后台增删、排序及配置图标

    2. 推荐列表

      • 推荐列表包含专辑资源
      • 推荐列表支持后台增删、排序及配置不同的专辑（包含专辑幅度在 3 - 10 之间）

    3. 专辑

      • 每张专辑至少包含 1 个音频文件
      • 专辑支持后台增删、音频排序、配置封面图片、修改专辑名字及作者

    M1-1-2-1. 乐点播 - 主页¶

    1. 点击 返回 按钮返回至 M1-1. 小乐迪设备主页
    2. 资源分类，点击分类进入 M1-1-2-2. 乐点播 - 分类列表
    3. 显示不同的推荐列表
    4. 显示推荐列表里面的专辑，包含专辑封面及专辑名称，可以左右滑动查看该推荐列表里面的所有专辑。点击 专辑 进入 M1-1-2-3. 乐点播 - 专辑页面
    5. 点击 设置 按钮进入  M1-1-2-14. 乐点播 - 主页 - 设置动作 ，并由于乐点播尚未改版，旧版主页见  M1-1-2-14-1. 乐点播 - 主页（旧版） 

    M1-1-2-2. 乐点播 - 分类列表¶

    1. 点击 返回 按钮返回至 M1-1-2-1. 乐点播 - 主页
    2. 显示 资源分类 标题，动态变化
    3. 显示 资源分类 下的所有专辑（分页加载 10 条 / 页），每条专辑显示专辑封面、专辑名字和作者，列表支持 下拉刷新
    4. 点击单条专辑列表进入 M1-1-2-3. 乐点播 - 专辑页面

    M1-1-2-3. 乐点播 - 专辑页面¶

    1. 点击 返回 按钮返回至前一个页面，区分 M1-1-2-1. 乐点播 - 主页 和 M1-1-2-2. 分类列表
    2. 显示专辑封面、专辑名字和作者
    3. 显示专辑内的音频列表，列表支持 下拉刷新。包含音频名字和音频时长。点击单条音频进行以下判断：
      • 若 MOB 已与 RBT 绑定，点击音频即可推送点播，见 M1-1-2-4. 乐点播 - 专辑页面 - 点播中
      • 若 MOB 未与 RBT 绑定，点击音频弹出绑定的提示框，见 M1-1-2-12. 乐点播 - 专辑页面 - 未绑定进行点播

    M1-1-2-4. 乐点播 - 专辑页面 - 点播中¶

    1. 进行推送点播中
      • 若点播成功，则进入 M1-1-2-5. 乐点播 - 专辑页面 - 歌曲点播中
      • 若乐迪不在线（网络断开），则进入  M1-1-2-6. 乐点播 - 专辑页面 - 乐迪离线 
      • 若手机离线，则进入  M1-1-2-13. 乐点播 - 点播失败 - 手机没有网络 

    M1-1-2-5. 乐点播 - 专辑页面 - 歌曲点播中¶

    1. 点击 返回 按钮返回至前一个页面，区分 M1-1-2-8. 乐点播 - 主页 - 歌曲点播中 和 M1-1-2-9. 乐点播 - 分类列表 - 歌曲点播中
    2. 显示专辑封面、专辑名字和作者
    3. 显示专辑内的音频列表，列表支持 下拉刷新。包含音频名字和音频时长，点击单条音频进行推送点播，见 M1-1-2-4. 乐点播 - 专辑页面 - 点播中。当前点播中的音频在曲目顺序位置显示动态播放标识
    4. 显示当前点播的专辑封面和曲目名字，MOB 需要知道 RBT 点播完毕，点播完毕后需要收起控制播放条
    5. 点击对点播歌曲进行 暂停 或 播放 的设定
    6. 点击对 RBT 的音量进行设置，见 M1-1-2-7. 乐点播 - 专辑页面 - 调节音量

    Note

    播放控制条 为乐点播全局通用

    M1-1-2-6. 乐点播 - 专辑页面 - 乐迪离线¶

    Failure

    此提示为警告 / 错误类 toast 提示

    1. 提示停留 3 秒之后收起
    2. 点击 关闭 按钮可以立即关闭提示，返回 M1-1-2-3. 乐点播 - 专辑页面

    M1-1-2-7. 乐点播 - 专辑页面 - 调节音量¶

    1. 点击进行音量大小调整
    2. 点击 音量 按钮关闭音量调节，返回 M1-1-2-5. 乐点播 - 专辑页面 - 歌曲点播中
    3. 点击屏幕任意位置关闭音量调节，返回 M1-1-2-5. 乐点播 - 专辑页面 - 歌曲点播中

    M1-1-2-8. 乐点播 - 主页 - 歌曲点播中¶

    1. 点击 返回 按钮返回至 M1-1. 小乐迪设备主页
    2. 资源分类，点击分类进入 M1-1-2-9. 乐点播 - 分类列表 - 歌曲点播中
    3. 显示不同的推荐列表
    4. 显示推荐列表里面的专辑，包含专辑封面及专辑名称，可以左右滑动查看该推荐列表里面的所有专辑。点击 专辑 进入 M1-1-2-5. 乐点播 - 专辑页面 - 歌曲点播中
    5. 音量播放控制条为乐点播功能全局通用，具体交互见 M1-1-2-5. 乐点播 - 专辑页面 - 歌曲点播中 和 M1-1-2-7. 乐点播 - 专辑页面 - 调节音量

    M1-1-2-9. 乐点播 - 分类列表 - 歌曲点播中¶

    1. 点击 返回 按钮返回至 M1-1-2-8. 乐点播 - 主页 - 歌曲点播中
    2. 显示 资源分类 标题，动态变化
    3. 显示 资源分类 下的所有专辑（分页加载 10 条 / 页），每条专辑显示专辑封面、专辑名字和作者，列表支持 下拉刷新
    4. 点击单条专辑列表进入 M1-1-2-4. 乐点播 - 专辑页面 - 点播中
    5. 音量播放控制条为乐点播功能全局通用，具体交互见 M1-1-2-5. 乐点播 - 专辑页面 - 歌曲点播中 和 M1-1-2-7. 乐点播 - 专辑页面 - 调节音量

    M1-1-2-10. 乐点播 - 无法加载¶

    Error

    当页面出现加载内容失败，出现此页面

    1. 点击 返回 按钮返回前一个页面
    2. 显示当前页面标题，动态变化
    3. 点击 点击刷新 按钮尝试重新加载页面，进入 M1-1-2-11. 乐点播 - 无法加载 - 加载中

    M1-1-2-11. 乐点播 - 无法加载 - 加载中¶

    1. 点击 返回 按钮返回前一个页面
    2. 显示当前页面标题，动态变化
    3. 尝试重新加载页面
      • 若加载成功则显示当前页面信息，页面内容根据实际动态变化
      • 加载失败继续显示加载失败的信息，返回至 M1-1-2-10. 乐点播 - 无法加载

    M1-1-2-12. 乐点播 - 专辑页面 - 未绑定进行点播¶

    Note

    当 MOB 未绑定 RBT 的时候进行点播操作，弹出此对话框

    1. 点击 去绑定 按钮进入绑定流程，见 M6-1. 小乐迪绑定流程
    2. 点击 取消 按钮关闭 alert，返回 M1-1-2-3. 乐点播 - 专辑页面

    M1-1-2-13. 乐点播 - 点播失败 - 手机没有网络¶

    1. 点击 我知道了 按钮关闭 Alert 并返回至  M1-1-2-3. 乐点播 - 专辑页面 

    M1-1-2-14. 乐点播 - 主页 - 设置动作¶

    1. 点击 关闭 按钮返回至  M1-1-2-1. 乐点播 - 主页 
    2. 点击 Switch 开关可以控制点播时乐迪是否跳舞，默认关闭，即默认状态下，点播歌曲小乐迪不会跳舞（定制歌曲不受影响，如《青春修炼手册》）

    M1-1-2-14-1. 乐点播 - 主页（旧版）¶

    Note

    和 M1-1-2-1. 乐点播 - 主页 交互一样

    M1-1-2-14-2. 乐点播 - 主页（旧版） - 设置动作¶

    Note

    和 M1-1-2-14. 乐点播 - 主页 - 设置动作 交互一样
