    M1-1-5-0. 小乐迪连网页面（手机没连 Wi-Fi）¶

    1. 点击 返回 按钮返回区分是否有绑定
      • 若 RBT 已被绑定，则返回至 M1-1. 小乐迪设备主页
      • 若 RBT 未被绑定，则返回至 M1-3. 小乐迪设备主页（未绑定）
      • 若从绑定流程跳转过来的，则返回至 M6-1. 绑定乐迪
    2. 显示当前手机设备没有连接任何 Wi-Fi 网络
      • 若是 iOS 设备，点击进行输入 Wi-Fi SSID 操作，见 M1-1-5-1-1. 小乐迪连网页面 - 填写 SSID（iOS）
      • 若是 Android 设备，点击进入搜索 Wi-Fi 操作，见 M1-1-5-2-1. 切换乐迪 Wi-Fi - 搜索中（Android）
    3. 点击进行输入 Wi-Fi 密码
    4. 由于没有填写连接任何 Wi-Fi 网络名称，因此此按钮不能点击
    5. 点击 连网引导 按钮进入 M1-1-5-7. 乐迪连网引导

    Tip

    连网流程区分 iOS 及 Android 两个版本，请同学们做好区分

    M1-1-5-1. 小乐迪连网页面（iOS）¶

    1. 点击 返回 按钮返回区分是否有绑定
      • 若 RBT 已被绑定，则返回至 M1-1. 小乐迪设备主页
      • 若 RBT 未被绑定，则返回至 M1-3. 小乐迪设备主页（未绑定）
      • 若从绑定流程跳转过来的，则返回至 M6-1. 绑定乐迪
    2. 显示当前手机设备所连接的 Wi-Fi 网络 SSID，点击可以进行修改，见 M1-1-5-1-1. 小乐迪连网页面 - 填写 SSID（iOS）
    3. 点击进行输入 Wi-Fi 密码，见 M1-1-5-1-2. 小乐迪连网页面 - 填写密码（iOS）
    4. 点击 连接网络 按钮将手机当前网络的 SSID 及密码保存，待发送至 RBT，见 M1-1-5-2. 连接乐迪的热点（iOS）
    5. 点击 连网引导 按钮进入 M1-1-5-7. 乐迪连网引导

    M1-1-5-1-1. 小乐迪连网页面 - 填写 SSID（iOS）¶

    1. 点击 返回 按钮返回区分是否有绑定
      • 若 RBT 已被绑定，则返回至 M1-1. 小乐迪设备主页
      • 若 RBT 未被绑定，则返回至 M1-3. 小乐迪设备主页（未绑定）
      • 若从绑定流程跳转过来的，则返回至 M6-1. 绑定乐迪
    2. 对 SSID 进行修改，点击 清空 按钮a可以一键清空 SSID
    3. 点击进行输入 Wi-Fi 密码，见 M1-1-5-1-2. 小乐迪连网页面 - 填写密码（iOS））
    4. 点击空白处收起键盘
    5. 点击 连网引导 按钮进入 M1-1-5-7. 乐迪连网引导

    M1-1-5-1-2. 小乐迪连网页面 - 填写密码（iOS）¶

    1. 点击 返回 按钮返回区分是否有绑定
      • 若 RBT 已被绑定，则返回至 M1-1. 小乐迪设备主页
      • 若 RBT 未被绑定，则返回至 M1-3. 小乐迪设备主页（未绑定）
      • 若从绑定流程跳转过来的，则返回至 M6-1. 绑定乐迪
    2. 显示当前手机设备所连接的 Wi-Fi 网络 SSID，点击可以进行修改，见 M1-1-5-1-1. 小乐迪连网页面 - 填写 SSID（iOS）
    3. Wi-Fi 密码默认 不可视，若不填写，默认此 Wi-Fi 是没有密码的开放性 Wi-Fi
    4. 点击空白处收起键盘
    5. 点击 连网引导 按钮进入 M1-1-5-7. 乐迪连网引导

    M1-1-5-2. 连接乐迪的热点（iOS）¶

    1. 点击 返回 按钮返回至 M1-1-5-1. 小乐迪连网页面（iOS）
    2. 图片截取自 iOS 手机提示用户连接 RBT 发射出来 AP 名称，AP 名字以最终生产版本为准
    3. AP 名字以最终生产版本为准
    4. 点击 去设置 Wi-Fi 按钮跳转到系统设置 Wi-Fi 页面，见 M1-1-5-3. 跳转到系统设置（iOS）
    5. 点击 连网引导 按钮进入 M1-1-5-7. 乐迪连网引导

    M1-1-5-3. 跳转到系统设置（iOS）¶

    1. 点击 iOS 10 系统自带的 back to app 按钮返回至乐迪机器人 app，见 M1-1-5-4. 等待机器人连网（iOS）
    2. 连接的 RBT AP 名字，AP 名字以最终生产版本为准

    M1-1-5-4. 等待机器人连网（iOS）¶

    1. 返回 MOB 后检测到已连接 AP 后，向 RBT 发送刚才保存的 Wi-Fi SSID 及密码资料，RBT 接收成功后由 AP 模式转变为正常模式，进行连网操作。此时 MOB 检测到断开 AP 后，进入 M1-1-5-5. 乐迪连网
      • 若没有连接 AP 就返回至 MOB，则进入 M1-1-5-2. 连接乐迪的热点（iOS）
      • 若返回 MOB 后超过 1 分钟 RBT 没有返回消息给 MOB，则提示 M1-1-5-10. 等待机器人连网 - 连接失败（iOS）

    Quote

    至此 iOS 与 Android 版本的区分已结束，快速前往 M1-1-5-5. 乐迪连网

    M1-1-5-1. 小乐迪连网页面（Android）¶

    1. 点击 返回 按钮返回区分是否有绑定
      • 若 RBT 已被绑定，则返回至 M1-1. 小乐迪设备主页
      • 若 RBT 未被绑定，则返回至 M1-3. 小乐迪设备主页（未绑定）
      • 若从绑定流程跳转过来的，则返回至 M6-1. 绑定乐迪
    2. 显示当前手机设备所连接的 Wi-Fi 网络 SSID，点击可以切换当前手机的网络，见 M1-1-5-2-1. 切换乐迪 Wi-Fi - 搜索中（Android）
    3. 点击进行输入 Wi-Fi 密码，见 M1-1-5-1-1. 小乐迪连网页面 - 填写密码（Android）
    4. 点击 连接网络 按钮将手机当前网络的 SSID 及密码保存，待发送至 RBT，见 M1-1-5-3 连接乐迪的热点 - 搜索中（Android）
    5. 点击 连网引导 按钮进入 M1-1-5-7. 乐迪连网引导

    M1-1-5-1-1. 小乐迪连网页面 - 填写密码（Android）¶

    1. 点击 返回 按钮返回区分是否有绑定
      • 若 RBT 已被绑定，则返回至 M1-1. 小乐迪设备主页
      • 若 RBT 未被绑定，则返回至 M1-3. 小乐迪设备主页（未绑定）
      • 若从绑定流程跳转过来的，则返回至 M6-1. 绑定乐迪
    2. 显示当前手机设备所连接的 Wi-Fi 网络 SSID，点击可以切换当前手机的网络，见 M1-1-5-2-1. 切换乐迪 Wi-Fi - 搜索中（Android）
    3. Wi-Fi 密码默认 不可视，若不填写，默认此 Wi-Fi 是没有密码的开放性 Wi-Fi
    4. 点击空白处收起键盘
    5. 点击 连网引导 按钮进入 M1-1-5-7. 乐迪连网引导

    M1-1-5-2-1. 切换乐迪 Wi-Fi - 搜索中（Android）¶

    1. 点击 取消 按钮返回至 M1-1-5-1. 小乐迪连网页面（Android）
    2. 进行搜索附近可用的 Wi-Fi 网络
      • 搜索有结果进入 M1-1-5-2-2. 切换乐迪 Wi-Fi（Android）
      • 搜索没有结果进入 M1-1-5-8. 切换手机网络 - 没有结果（Android）

    M1-1-5-2-2. 切换乐迪 Wi-Fi（Android）¶

    1. 点击 取消 按钮返回至 M1-1-5-1. 小乐迪连网页面（Android）
    2. 点击任意网络返回至 M1-1-5-1. 小乐迪连网页面（Android） 的同时把 SSID 保存为当前连接网络 SSID

    M1-1-5-3 连接乐迪的热点 - 搜索中（Android）¶

    1. 点击 返回 按钮返回至 M1-1-5-1. 小乐迪连网页面（Android）
    2. 扫描 RBT 生成的 AP 热点，过滤其他 Wi-Fi 和手机热点
      • 扫描有结果进入 M1-1-5-3-1. 连接乐迪的热点（Android））
      • 1 分钟超时后没有发现任何 RBT 的 AP，则进入 M1-1-5-9. 连接乐迪的热点 - 没有结果（Android）
    3. 点击 扫描 按钮进行重新扫描
    4. 点击 连网引导 按钮进入 M1-1-5-7. 乐迪连网引导

    M1-1-5-3-1. 连接乐迪的热点（Android）¶

    1. 点击 返回 按钮返回至 M1-1-5-1. 小乐迪连网页面（Android）
    2. 点击 RBT 发射出来 AP 名称（AP 名字以最终生产版本为准）进行连接，进入 M1-1-5-4. 等待机器人连网（Android）
    3. 点击 扫描 按钮进行重新扫描，见 M1-1-5-3 连接乐迪的热点 - 搜索中（Android）
    4. 点击 连网引导 按钮进入 M1-1-5-7. 乐迪连网引导

    1. 返回 MOB 后检测到已连接 AP 后，向 RBT 发送刚才保存的 Wi-Fi SSID 及密码资料，RBT 接收成功后由 AP 模式转变为正常模式，进行连网操作。此时 MOB 检测到断开 AP 后，进入 M1-1-5-5. 乐迪连网
      • 若没有连接 AP 就返回至 MOB，则进入 M1-1-5-3-1. 连接乐迪的热点（Android）
      • 若返回 MOB 后超过 1 分钟 RBT 没有返回消息给 MOB，则提示 M1-1-5-9. 连接乐迪的热点 - 没有结果（Android）

    至此 iOS 与 Android 版本的区分已结束，快速前往 M1-1-5-5. 乐迪连网

    M1-1-5-5. 乐迪连网¶

    Notes

    此处 RBT 会通过语音提示用户该如何进行下一步操作

    1. 点击 连网成功 按钮进入 M1-1-5-6. 绑定乐迪
    2. 点击 重新设置 按钮根据手机型号返回至 M1-1-5-1. 小乐迪连网页面（iOS） 或 M1-1-5-1. 小乐迪连网页面（Android）
    3. 点击 连网引导 按钮进入 M1-1-5-7. 乐迪连网引导

    M1-1-5-6. 绑定乐迪¶

    1. 点击 返回 按钮返回至 M1-1-5-5. 乐迪连网
    2. 显示当前乐迪连接的 Wi-Fi
    3. 点击 绑定乐迪 按钮进入绑定流程，见 M6-2. 扫描二维码
    4. 点击 随便看看 按钮进入 M1-3. 小乐迪设备主页（未绑定）

    M1-1-5-7. 乐迪连网引导¶

    1. 点击 返回 按钮返回前一个页面
    2. 用一个 HTML 5 的页面显示对应的引导内容，区分 iOS 和 Android 版本。如内容出现加载失败的情况，则进入 M1-1-5-7-1. 乐迪连网引导 - 无法加载

    M1-1-5-7-1. 乐迪连网引导 - 无法加载¶

    Error

    当页面出现加载内容失败，出现此页面

    1. 点击 返回 按钮返回前一个页面
    2. 点击 点击刷新 按钮尝试重新加载页面，进入 M1-1-5-7-2. 乐迪连网引导 - 无法加载 - 加载中

    M1-1-5-7-2. 乐迪连网引导 - 无法加载 - 加载中¶

    1. 点击 返回 按钮返回前一个页面
    2. 尝试重新加载页面
      • 若加载成功则显示当前页面信息，见 M1-1-5-7. 乐迪连网引导
      • 加载失败继续显示加载失败的信息，返回至 M1-1-5-7-1. 乐迪连网引导 - 无法加载

    M1-1-5-8. 切换手机网络 - 没有结果（Android）¶

    Note

    当手机搜索后没有发现可用 Wi-Fi 网络后提示

    1. 点击 取消 按钮返回至 M1-1-5-1. 小乐迪连网页面（Android）
    2. 提示附近没有可用的 Wi-Fi 网络

    M1-1-5-9. 连接乐迪的热点 - 没有结果（Android）¶

    Note

    当手机搜索后没有发现 RBT 生成的 AP 热点后提示

    1. 点击 返回 按钮返回至 M1-1-5-1. 小乐迪连网页面（Android）
    2. 提示没有发现 RBT 生成的热点
    3. 点击 扫描 按钮进行重新扫描，见 M1-1-5-3 连接乐迪的热点 - 搜索中（Android）
    4. 点击 连网引导 按钮进入 M1-1-5-7. 乐迪连网引导
