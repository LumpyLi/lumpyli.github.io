    M4. 我的¶

    1. 显示宝贝昵称，点击进入 M4-1. 我的宝贝
    2. 显示家长昵称，点击进入 M4-2. 家长帐号
    3. 点击进入 M4-3. 我的客服
    4. 点击进行清除缓存操作，见 M4-4. 清除缓存
    5. 点击进入 M4-5. 关于我们

    7. 点击进入 M2. 发现
    8. 点击进入 M3. 商城

    M4-0. 我的 - 未绑定宝贝¶

    Note

    当用户未绑定宝贝时，隐藏宝贝信息入口

    1. 显示家长昵称，点击进入 M4-2. 家长帐号
    2. 点击进入 M4-3. 我的客服
    3. 点击进行清除缓存操作，见 M4-4. 清除缓存
    4. 点击进入 M4-5. 关于我们
    5. 击进入 M1-1. 小乐迪设备主页
    6. 点击进入 M2. 发现
    7. 点击进入 M3. 商城

    M4-1. 我的宝贝¶

    1. 点击 返回 按钮返回至 M4.我的
    2. 显示默认 宝贝头像 （此版本暂不支持更换）
    3. 显示宝贝信息，包括性别及年龄，年龄格式为 2 岁 3 个月 或 4 岁
    4. 点击 编辑 按钮进入  M4-1-1. 宝贝帐号 
    5. 显示小乐迪（已绑定状态）的信息，包括 设备昵称 和 产品名，点击进入  M4-1-7. 小乐迪设备详情 
      • 若此类型设备未被绑定，则进入 M6-1. 小乐迪绑定流程
    6. 显示大乐迪（未绑定状态）的信息，包括 设备昵称 和 产品名，点击进入 M6-2. 大乐迪绑定流程
      • 若此类型设备已被绑定，则进入  M4-1-8. 大乐迪设备详情 

    M4-1-1. 宝贝帐号¶

    1. 点击 返回 按钮返回至 M4-1. 我的宝贝
    2. 显示默认宝贝头像（此版本暂不支持更换）
    3. 点击进入  M4-1-2. 宝贝帐号 - 宝贝昵称  进行修改宝贝昵称
    4. 点击进入  M4-1-3. 宝贝帐号 - 选择生日  进行修改宝贝生日
    5. 根据生日计算出宝贝年龄，年龄格式为 2 岁 3 个月 或 4 岁
    6. 点击进入  M4-1-4. 宝贝帐号 - 选择性别  进行修改宝贝性别
    7. 点击进入  M4-1-5. 宝贝帐号 - 宝贝关系  进行修改与宝贝的关系

    M4-1-2. 宝贝帐号 - 宝贝昵称¶

    1. 点击 返回 按钮返回至  M4-1-1. 宝贝帐号 
    2. 当宝贝昵称有输入至少 1 个字符后为可点击状态，点击后保存昵称并返回至  M4-1-1. 宝贝帐号 
      • 若输入超过 8 个字符，见  M4-1-2-1. 宝贝帐号 - 宝贝昵称超长 
      • 若输入非汉字、英文和数字（如空格、标点符号、下划线等），见  M4-1-2-2. 宝贝帐号 - 宝贝昵称含有不合法字符 
    3. 进入页面后，光标定位在 Text Field 上并弹出键盘
    4. 点击可以清空 Text Field

    M4-1-2-1. 宝贝帐号 - 宝贝昵称超长¶

    Failure

    此提示为警告 / 错误类 toast 提示

    1. 提示停留 3 秒之后收起
    2. 点击 关闭 按钮可以立即关闭提示，返回  M4-1-1. 宝贝帐号 

    M4-1-2-2. 宝贝帐号 - 宝贝昵称含有不合法字符¶

    Failure

    此提示为警告 / 错误类 toast 提示

    1. 提示停留 3 秒之后收起
    2. 点击 关闭 按钮可以立即关闭提示，返回  M4-1-1. 宝贝帐号 

    M4-1-3. 宝贝帐号 - 选择生日¶

    Quote

    系统要根据宝贝生日计算 宝贝年龄，格式是 x 岁 x 个月 / x 岁，例如 3 岁 1 个月 / 4 岁

    1. Date Picker 默认日期是 2014 年 1 月 1 日，可选范围是 1970 年 1 月 1 日至当前日期，超出范围不可选
    2. 点击 完成 按钮保存日期到宝贝生日，并关闭当前的 Date Picker
    3. 点击遮罩 不保存 日期到宝贝生日，并关闭当前的 Date Picker

    M4-1-4. 宝贝帐号 - 选择性别¶

    1. 选择 男孩 保存到宝贝性别，并关闭当前的 Action Sheet
    2. 选择 女孩 保存到宝贝性别，并关闭当前的 Action Sheet
    3. 点击 取消 按钮不保存性别到宝贝性别，并关闭当前的 Action Sheet
    4. 点击遮罩层不保存性别到宝贝性别，并关闭当前的 Action Sheet

    M4-1-5. 宝贝帐号 - 宝贝关系¶

    1. 点击 返回 按钮返回至 M4-1-1. 宝贝帐号
    2. 当输入有效的宝贝关系后，保存 按钮变为可点击状态，点击后保存数据并返回至  M4-1-1. 宝贝帐号 
      • 若超出 12 个字符，则在光标离开输入框的时候提示  M4-1-5-1. 宝贝帐号 - 宝贝关系超长 
      • 若含有非汉字、英文和数字（如空格、标点符号、下划线等），则在光标离开输入框的时候提示  M4-1-5-2. 宝贝帐号 - 宝贝关系含有不合法字符 
    3. 光标自动定位输入框并弹出键盘，可输入 1 - 12 个字符，不可含有非汉字、英文和数字（如空格、标点符号、下划线等）
    4. 点击进行快速选择，选择后 Text Field 自动变更成对应的关系
      • 若修改成自定义关系，则快速选择的选项不被选中
