    M5. 宝贝设备

    What's New?¶

    2017-05-10¶

    • 首次提交

    Spec¶

    Note

    下载全部 Spec 图片 Update: 2017-05-10

    M5-1. 宝贝设备¶

    1. 点击 返回 按钮返回至 M1-1. 小乐迪设备主页
    2. 显示当前性别对应的默认头像，待支持自定义宝贝头像才提供修改功能
    3. 显示宝贝的昵称、性别及年龄
    4. 点击 修改 按钮进行对宝贝信息的修改，见 M5-2. 宝贝账号
    5. 点击 添加 按钮可以绑定新的设备，见 M5-3. 添加设备
    6. 显示当前只绑定了小乐迪一台设备，包括设备的图片、设备的昵称及设备的产品名，点击可进入 M5-5. 小乐迪设备详情

    M5-2. 宝贝帐号¶

    1. 点击 返回 按钮返回至 M5-1. 宝贝设备
    2. 显示当前性别对应的默认头像，待支持自定义宝贝头像才提供修改功能
    3. 点击进入修改宝贝的昵称，见 M5-2-1. 宝贝账号 - 宝贝昵称
    4. 点击进入修改宝贝的生日，见 M5-2-2. 宝贝账号 - 选择生日
    5. 根据宝贝生日计算宝贝年龄，格式是 x 岁 x 个月 / x 岁，例如 3 岁 1 个月 / 4 岁
    6. 点击进入修改宝贝的性别，同时更换默认头像，见 M5-2-3. 宝贝账号 - 选择性别
    7. 点击进入修改与宝贝的关系，见 M5-2-4. 宝贝账号 - 宝贝关系

    M5-2-1. 宝贝帐号 - 宝贝昵称¶

    1. 点击 返回 按钮不保存修改直接返回至 M5-2. 宝贝账号
    2. 当昵称发生了修改且昵称不为空时，保存 按钮才可点击，点击后保存数据并返回至 M5-2. 宝贝账号，并进行以下判断：
      • 昵称是否超出 8 个字符的限制，错误提示见 M6-6-4. 添加宝贝 - 填写昵称 - 宝贝昵称超长
      • 昵称是否含有空格或标点符号等不合法字符，错误提示见 M6-6-5. 添加宝贝 - 填写昵称 - 宝贝昵称含有不合法字符
    3. 进入此页面时光标自动定位并弹出键盘
    4. 点击 清空 按钮清空当前 Text Field 的内容

    M5-2-2. 宝贝帐号 - 选择生日¶

    Quote

    系统要根据宝贝生日计算 宝贝年龄，格式是 x 岁 x 个月 / x 岁，例如 3 岁 1 个月 / 4 岁

    1. 点击 完成 按钮保存当前日期到宝贝生日，并关闭当前的 Date Picker
    2. Date Picker 可选范围是 1970 年 1 月 1 日至当前日期，超出范围不可选
    3. 点击遮罩 不保存 当前日期到宝贝生日，并关闭当前的 Date Picker

    M5-2-3. 宝贝帐号 - 选择性别¶

    1. 选择 男孩 保存到宝贝性别，并关闭当前的 Action Sheet
    2. 选择 女孩 保存到宝贝性别，并关闭当前的 Action Sheet
    3. 点击 取消 按钮不保存性别到宝贝性别，并关闭当前的 Action Sheet
    4. 点击遮罩层不保存性别到宝贝性别，并关闭当前的 Action Sheet

    M5-2-4. 宝贝帐号 - 宝贝关系¶

    1. 点击 返回 按钮不保存修改直接返回至 M5-2. 宝贝账号
    2. 当输入有效的宝贝关系后，保存 按钮变为可点击状态，并保存数据到 M5-2. 宝贝帐号，同时进行以下判断：
      • 若超出 12 个字符，错误提示见 M6-6-6. 添加宝贝 - 选择与宝贝关系 - 输入自定义关系 - 宝贝关系超长
      • 若含有空格或标点符号，错误提示见 M6-6-7. 添加宝贝 - 选择与宝贝关系 - 输入自定义关系 - 宝贝关系含有不合法字符
    3. 可输入 1 - 12 个字符，不可含有空格或标点符号
    4. 点击快速选择与宝贝的关系，单选，选中后在 Text Field 上显示对应关系，不能手动取消选中。当 Text Field 发生自定义修改时，快速选中项会取消选中

    M5-3. 添加设备¶

    Note

    由于目前仅支持一个产品型号绑定一台设备，因此在已绑定一台小乐迪的情况下，添加 乐迪益智机器人 的按钮不能被点击

    1. 点击 关闭 按钮关闭页面，返回至 M5-1. 宝贝设备
    2. 点击进行添加大乐迪，见 M6-2. 大乐迪绑定流程
    3. 点击进行添加小乐迪，见 M6-1. 大乐迪绑定流程

    M5-4. 宝贝设备 - 齐全¶

    Note

    此页面系绑定了小乐迪及大乐迪时显示的状态

    1. 点击 返回 按钮返回至 M1-1. 小乐迪设备主页
    2. 显示当前性别对应的默认头像，待支持自定义宝贝头像才提供修改功能
    3. 显示宝贝的昵称、性别及年龄
    4. 点击 修改 按钮进行对宝贝信息的修改，见 M5-2. 宝贝账号
    5. 点击 添加 按钮可以绑定新的设备，见 M5-3. 添加设备
    6. 显示当前控制小乐迪中，点击可进入 M5-5. 小乐迪设备详情
    7. 显示已绑定一台大乐迪，点击可进入 M5-6. 大乐迪设备详情

    M5-5. 小乐迪设备详情¶

    1. 点击 返回 按钮返回至 M5-1. 宝贝设备
    2. 当前控制小乐迪中，因此 切换 按钮不可被点击
    3. 点击对设备昵称进行修改，见 M5-7. 设备昵称
    4. 点击 解除绑定 按钮解除与此设备的绑定，进入 M5-9. 解除绑定中

    M5-6. 大乐迪设备详情¶

    1. 点击 返回 按钮返回至 M5-1. 宝贝设备
    2. 当前控制小乐迪中，因此 切换 按钮可被点击，点击进入 M5-8. 切换设备中
    3. 点击对设备昵称进行修改，见 M5-7. 设备昵称
    4. 点击 解除绑定 按钮解除与此设备的绑定，进入 M5-9. 解除绑定中

    M5-7. 设备昵称¶

    1. 点击 返回 按钮不对修改进行保存，并返回至 M5-5. 小乐迪设备详情
    2. 当昵称发生了修改且昵称不为空时，保存 按钮才可点击，点击后保存数据并返回至 M5-5. 小乐迪设备详情，并进行以下判断：
      • 昵称是否超出 20 个字符的限制，错误提示见 M5-7-1. 设备昵称超长
    3. 进入此页面时光标自动定位并弹出键盘
    4. 点击 清空 按钮清空当前 Text Field 的内容

    M5-7-1. 设备昵称超长¶

    Failure

    此提示为警告 / 错误类 toast 提示

    1. 提示停留 3 秒之后收起
    2. 点击 关闭 按钮可以立即关闭提示，返回 M5-7. 设备昵称

    M5-8. 切换设备中¶

    1. 切换控制当前设备，并进行以下判断：
      • 若检测到设备未被解绑，则进入对应的设备主页，如 M1-1. 小乐迪设备主页 或 M1-3. 大乐迪设备主页
      • 若检测到设备已被解绑，则进入 M5-10. 切换时发现已被解绑

    M5-9. 解除绑定中¶

    1. 对此设备进行解绑操作，并进行以下判断：
      • 若解绑后还有其他设备在绑，则进入 M5-11-1. 解除绑定成功（还有设备）
      • 若解绑后没有其他设备在绑，则切换至最近被绑定的一台设备，并进入 M5-11-2. 解除绑定成功（已无设备）

    M5-10. 切换时发现已被解绑¶

    1. 点击 我知道了 按钮关闭对话框，并返回至 M5-1. 宝贝设备

    M5-11-1. 解除绑定成功（还有设备）¶

    Success

    此提示为通知类 toast 提示

    1. 提示停留 3 秒之后收起
    2. 点击 关闭 按钮可以立即关闭提示，返回 M5-1. 宝贝设备

    M5-11-2. 解除绑定成功（已无设备）¶

    Success

    此提示为通知类 toast 提示

    1. 提示停留 3 秒之后收起
    2. 点击 关闭 按钮可以立即关闭提示，返回 M1-3. 小乐迪设备主页（未绑定）
