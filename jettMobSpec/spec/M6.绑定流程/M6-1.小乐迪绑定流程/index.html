    Spec¶

    M6-1. 绑定乐迪¶

    1. 点击 返回 按钮返回至 M1-3. 小乐迪设备主页（未绑定）
    2. 点击 绑定乐迪 按钮进入 M6-2. 扫描二维码
    3. 点击 设备还没有网络？ 按钮进入 M1-1-5-0. 小乐迪连网页面（手机没连 Wi-Fi） 进入连网流程

    M6-2. 扫描二维码¶

    1. 点击 返回 按钮返回至 M6-1. 绑定乐迪
    2. 识别到是二维码后进入 M6-2-1. 扫一扫（扫描中）

    M6-2-1. 扫描二维码（扫描中）¶

    1. 根据不同的二维码跳转不同的页面
      • 若扫描时，手机网络不可用，则进入 M6-6-1. 全局网络不可用
      • 若识别到是 RBT 生成的绑定二维码，则进入 M6-3. 添加宝贝
      • 若识别到是第三方二维码，则进入 M6-6-2. 扫一扫（扫描第三方二维码）
      • 若二维码超时失效（有效时间 3 分钟），则进入 M6-6-3. 扫一扫（OTT 超时）

    M6-3. 添加宝贝¶

    1. 点击 取消 按钮退出添加宝贝流程，返回至 M0-1.登录 or 注册页面
    2. 点击输入宝贝昵称，见 M6-3-1. 添加宝贝 - 填写昵称
    3. 点击选择宝贝生日，见 M6-3-2. 添加宝贝 - 选择生日
    4. 点击选择宝贝性别，见 M6-3-3. 添加宝贝 - 选择性别
    5. 点击选择与宝贝的关系，见 M6-3-4. 添加宝贝 - 选择与宝贝关系
    6. 点击输入自定义与宝贝的关系，见 M6-3-5. 添加宝贝 - 选择与宝贝关系 - 输入自定义关系
    7. 当所有数据填写完毕，添加宝贝 按钮变为可点击状态。点击之后进入添加宝贝头像，见 M6-4. 添加宝贝头像

    M6-3-1. 添加宝贝 - 填写昵称¶

    1. 点击 返回 按钮返回至 M6-3. 添加宝贝
    2. 弹出输入法键盘，可输入 1 - 8 个字符，不可含有空格或标点符号
      • 若超出 8 个字符，则在光标离开输入框的时候提示 M6-6-4. 添加宝贝 - 填写昵称 - 宝贝昵称超长
      • 若含有空格或标点符号，则在光标离开输入框的时候提示 M6-6-5. 添加宝贝 - 填写昵称 - 宝贝昵称含有不合法字符
    3. 点击选择宝贝生日，见 M6-3-2. 添加宝贝 - 选择生日
    4. 点击选择宝贝性别，见 M6-3-3. 添加宝贝 - 选择性别
    5. 点击选择与宝贝的关系，见 M6-3-4. 添加宝贝 - 选择与宝贝关系
    6. 点击 return 按钮或屏幕空白位置收起键盘

    M6-3-2. 添加宝贝 - 选择生日¶

    Quote

    系统要根据宝贝生日计算 宝贝年龄，格式是 x 岁 x 个月 / x 岁，例如 3 岁 1 个月 / 4 岁

    1. Date Picker 默认日期是 2014 年 1 月 1 日，可选范围是 1970 年 1 月 1 日至当前日期，超出范围不可选
    2. 点击 完成 按钮保存日期到宝贝生日，并关闭当前的 Date Picker
    3. 点击遮罩 不保存 日期到宝贝生日，并关闭当前的 Date Picker

    M6-3-3. 添加宝贝 - 选择性别¶

    1. 选择 男孩 保存到宝贝性别，并关闭当前的 Action Sheet
    2. 选择 女孩 保存到宝贝性别，并关闭当前的 Action Sheet
    3. 点击 取消 按钮不保存性别到宝贝性别，并关闭当前的 Action Sheet
    4. 点击遮罩层不保存性别到宝贝性别，并关闭当前的 Action Sheet

    M6-3-4. 添加宝贝 - 选择与宝贝关系¶

    1. 点击 取消 按钮退出添加宝贝流程，返回至 M0-1.登录 or 注册页面
    2. 点击输入宝贝昵称，见 M6-3-1. 添加宝贝 - 填写昵称
    3. 点击选择宝贝生日，见 M6-3-2. 添加宝贝 - 选择生日
    4. 点击选择宝贝性别，见 M6-3-3. 添加宝贝 - 选择性别
    5. 点击快速选择与宝贝的关系，单选，选中后不能取消
    6. 点击输入自定义与宝贝的关系，见 M6-3-5. 添加宝贝 - 选择与宝贝关系 - 输入自定义关系
    7. 当所有数据填写完毕，添加宝贝 按钮变为可点击状态。点击之后进入添加宝贝头像，见 M6-4. 添加宝贝头像

    M6-3-5. 添加宝贝 - 选择与宝贝关系 - 输入自定义关系¶

    1. 点击 取消 按钮不保存数据并返回至 M6-3. 添加宝贝
    2. 当输入有效的宝贝关系后，保存 按钮变为可点击状态，点击后保存数据并返回至 M6-3-6. 添加宝贝 - 显示自定义关系
      • 若超出 12 个字符，则在光标离开输入框的时候提示 M6-6-6. 添加宝贝 - 选择与宝贝关系 - 输入自定义关系 - 宝贝关系超长
      • 若含有空格或标点符号，则在光标离开输入框的时候提示 M6-6-7. 添加宝贝 - 选择与宝贝关系 - 输入自定义关系 - 宝贝关系含有不合法字符
    3. 光标自动定位输入框并弹出键盘，可输入 1 - 12 个字符，不可含有空格或标点符号
    4. 点击 return 按钮作用同 保存 按钮

    M6-3-6. 添加宝贝 - 显示自定义关系¶

    1. 点击 取消 按钮退出添加宝贝流程，返回至 M0-1.登录 or 注册页面
    2. 点击输入宝贝昵称，见 M6-3-1. 添加宝贝 - 填写昵称
    3. 点击选择宝贝生日，见 M6-3-2. 添加宝贝 - 选择生日
    4. 点击选择宝贝性别，见 M6-3-3. 添加宝贝 - 选择性别
    5. 点击选择与宝贝的关系，见 M6-3-4. 添加宝贝 - 选择与宝贝关系
    6. 默认显示 4 个字符，超过 4 个字符之后用 前 4 个字符 + ... 代替。点击输入自定义与宝贝的关系，见 M6-3-5. 添加宝贝 - 选择与宝贝关系 - 输入自定义关系
    7. 当所有数据填写完毕，添加宝贝 按钮变为可点击状态。点击之后进入添加宝贝头像，见 M6-4. 添加宝贝头像

    M6-4. 添加宝贝头像¶

    Quote

    宝贝头像是选填项目

    1. 点击 返回 按钮返回至 M6-3. 添加宝贝
    2. 点击进入 M6-4-1. 添加宝贝头像 - 选择添加头像方式
    3. 显示 M6-3-1. 添加宝贝 - 填写昵称 所填写的宝贝昵称
    4. 点击 开始使用 按钮进入 M6-5. 激活中

    M6-4-1. 添加宝贝头像 - 选择添加头像方式¶

    Quote

    照片需要进行圆形剪裁

    1. 点击 选取照片 按钮进入系统相册
    2. 点击 拍照 按钮进入系统相机
    3. 点击 取消 按钮关闭 Action Sheet，返回至 M6-4. 添加宝贝头像
    4. 点击遮罩层关闭 Action Sheet，返回至 M6-4. 添加宝贝头像

    M6-5. 激活中¶

    1. 分以下两种情况：
      • 若激活成功，则进入 M1-1. 小乐迪设备主页
      • 若因为手机网络原因导致激活失败，则进入 M6-6-1. 全局网络不可用

    M6-6-1. 全局网络不可用¶

    Note

    此对话框为全局 alert，当由于网络原因造成手机无法与服务器进行交互的时候，弹出此对话框

    1. 点击 我知道了 按钮关闭对话框并返回前一个页面

    M6-6-2. 扫一扫（扫描第三方二维码）¶

    Failure

    此提示为警告 / 错误类 toast 提示

    1. 提示停留 3 秒之后收起
    2. 点击 关闭 按钮可以立即关闭提示，返回 M6-2. 扫描二维码

    M6-6-3. 扫一扫（OTT 超时）¶

    Note

    此对话框为全局 alert，当由于 OTT 超时导致二维码失效的时候，弹出此对话框

    1. 点击 我知道了 按钮返回 M6-2. 扫描二维码
