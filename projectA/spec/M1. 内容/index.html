    M1. 内容

    Note

    低保真 UI Flow 在蓝湖，点击查看

    Important

    内容分别由 阿里 及 奥睿 两家提供，由于 AliGenie SDK 的限制，在阿里服务器内的内容才能进行 收藏 操作。

    因此：

    乐迪电台、妈妈讲故事、我的上传三个模块下的内容 没有收藏功能

    M1-0. 内容首页 - 全¶

    1. 显示四个分类，由阿里提供内容。分别是 早教故事、儿歌点唱、英语启蒙 和 国学经典，点击分类进入 M1-3. 内容 - 分类列表 - 专辑列表
    2. 乐迪电台，由奥睿提供内容，以 儿歌 为主，点击 立即收听 按钮进行点播推送给设备，推送的内容为随机且连续播放，直到用户点击停止
      • 若用户未绑定设备，则进入 MX-2. 配网
      • 若设备没有在线，则进入 M1-4-3. 内容 - 发送点播指令 - 点播失败 - 设备没连网
      • 若手机没有网络，则进入 M1-4-4. 内容 - 发送点播指令 - 点播失败 - 手机无网络
      • 点播成功见 M1-1-1. 内容首页 - 点播歌曲中
    3. 睡前故事，由阿里提供内容，以 故事 为主，点击方块进行点播推送给设备，推送的内容为随机且连续播放，直到用户点击停止
      • 若用户未绑定设备，则进入 MX-2. 配网
      • 若设备没有在线，则进入 M1-4-3. 内容 - 发送点播指令 - 点播失败 - 设备没连网
      • 若手机没有网络，则进入 M1-4-4. 内容 - 发送点播指令 - 点播失败 - 手机无网络
      • 点播成功见 M1-1-1. 内容首页 - 点播歌曲中
    4. 妈妈讲故事，让用户可以对照故事模版进行录音，点击进入 M1-7-1. 妈妈讲故事
    5. 以专辑组合的推荐，名称可变
      • 一次显示 5 - 10 个
    6. 点击 查看更多 按钮显示当前推荐所有专辑，参考 M1-3. 内容 - 分类列表 - 专辑列表
    7. 显示专辑封面及名称，点击进入至 M1-4. 内容 - 专辑页面

    M1-0-1. 全局 - 加载失败 / 手机没网络¶

    1. 点击 重新加载 按钮对页面重新加载，见 M1-0-2. 全局 - 重新加载

    1. 尝试进行重新加载
      • 若加载成功，则进入 M1-1. 内容首页
      • 若超过 30 秒还是加载失败，则返回至 M1-0-1. 全局 - 加载失败 / 手机没网络

    M1-1-1. 内容首页 - 点播歌曲中¶

    Note

    当在点播时，播放控制器在 内容 页全局显示

    1. 显示 专辑封面
    2. 显示 音频标题 及 专辑名称
    3. 收藏 按钮，点击可进行收藏，见 M1-5-2. 收藏音频
    4. 点击进行切换下一首
    5. 点击可展开控制面板，见 M1-5-1. 播放器 - 展开

    M1-3. 内容 - 分类列表 - 专辑列表¶

    1. 点击 返回 按钮返回至 M1-1. 内容首页
    2. 根据分类内容显示不同的引导语
    3. 显示当前分类下的 专辑 及 专辑名

    M1-3-1. 内容 - 分类列表 - 专辑列表 - 点播中¶

    Note

    当在点播时，播放控制器在 内容 页全局显示

    M1-4. 内容 - 专辑页面¶

    1. 点击 返回 按钮返回至上一级
    2. 显示 专辑封面 以及 专辑名称
    3. 显示当前专辑名的引导语
    4. 点击 播放全部 按钮对当前专辑进行点播，见 M1-4-1. 内容 - 发送点播指令
      • 由第一首开始点播
      • 按顺序连续播点播
      • 循环点播直至用户点击停止
    5. 显示当前专辑包含的曲目数量
    6. 点击任意一首曲目可以点播当前曲目
      • 由当前曲目开始点播，见 M1-4-1. 内容 - 发送点播指令
      • 按顺序连续播点播
      • 循环点播直至用户点击停止

    M1-4-1. 内容 - 发送点播指令¶

    1. 进行点播操作
      • 若点播成功，则进入 M1-4-2. 内容 - 发送点播指令 - 点播中
      • 若设备没有在线，则进入 M1-4-3. 内容 - 发送点播指令 - 点播失败 - 设备没连网
      • 若手机没有网络连接，则进入 M1-4-4. 内容 - 发送点播指令 - 点播失败 - 手机无网络

    M1-4-2. 内容 - 发送点播指令 - 点播中¶

    Note

    当在点播时，播放控制器在 内容 页全局显示

    1. 当前点播曲目

    M1-4-3. 内容 - 发送点播指令 - 点播失败 - 设备没连网¶

    Failure

    此提示为警告 / 错误类 Toast 提示

    1. 提示停留 3 秒之后收起，返回至 M1-4. 内容 - 专辑页面

    M1-4-4. 内容 - 发送点播指令 - 点播失败 - 手机无网络¶

    1. 点击 知道了 按钮返回至 M1-4. 内容 - 专辑页面

    M1-5-1. 播放器 - 展开¶

    Note

    当在点播时，播放控制器在 内容 页全局显示

    1. 显示 专辑封面
    2. 显示 音频标题 及 专辑名称
    3. 收藏 按钮，点击可进行收藏，见 M1-5-2. 收藏音频
    4. 点击可收起控制面板
    5. 点击切换 上一首 音频
    6. 点击 停止播放 音频，隐藏控制面板，返回至 M1-4. 内容 - 专辑页面
    7. 点击切换 下一首 音频
    8. 点击 睡眠模式 按钮进入 M1-6. 播放器 - 睡眠模式

    M1-5-2. 收藏音频¶

    Note

    当在点播时，播放控制器在 内容 页全局显示

    1. 显示收藏成功，且将此音频添加至 我的收藏 播放列表，并在 内容 首页显示此播放列表，见 M1-9-1. 收藏列表 - 首页入口 - 全
      • 若手机没有网络导致不能收藏，弹出 手机没有网络 的错误提示，参见 M1-4-4. 内容 - 发送点播指令 - 点播失败 - 手机无网络
    2. 控制面板上的 收藏 按钮激活，再次点击就会取消收藏当前音频

    M1-6. 播放器 - 睡眠模式¶

    1. 选择睡眠模式的时长，分别有 15 分钟、30 分钟、45 分钟、60 分钟四种，选择一个时间进入 M1-6-1. 播放器 - 睡眠模式 - 睡眠播放中
    2. 点击 取消 按钮收起 Action Sheet 返回至前一个界面

    M1-6-1. 播放器 - 睡眠模式 - 睡眠播放中¶

    1. 点击 退出 按钮弹出对话框，见 M1-6-2. 播放器 - 睡眠模式 - 退出睡眠模式
    2. 默认开启设备的小夜灯，点击可关闭小夜灯，见 M1-6-3. 播放器 - 睡眠模式 - 关闭小夜灯 - 提示
    3. 点击 停止 按钮暂停播放音频，但仍会在睡眠模式下
    4. 点击切换 下一首 音频
    5. 显示当前剩余的分钟数，以 整数 显示，点击弹出选择时间的 Action Sheet 重新设定时间，见 M1-6. 播放器 - 睡眠模式
    6. 显示当前点播中的 曲目名字

    M1-6-2. 播放器 - 睡眠模式 - 退出睡眠模式¶

    1. 点击 取消 按钮关闭 Alert，返回至 M1-6-1. 播放器 - 睡眠模式 - 睡眠播放中
    2. 点击 退出哄睡 按钮关闭 Alert 并退出睡眠模式，返回至进来睡眠模式前的一个界面
      • 关闭设备的小夜灯
      • 停止播放音频
