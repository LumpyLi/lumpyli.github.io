    MX-1. 登录

    Note

    低保真 UI Flow 在蓝湖，点击查看

    MX-1-1. 登录前页面 - 未勾选¶

    1. Checkbox 默认未选中，选中后见 MX-1-2. 登录前页面 - 已勾选；点击 用户协议 进入 MX-2-1. 用户协议
    2. 开始 按钮默认 Disabled 状态

    MX-1-2. 登录前页面 - 已勾选¶

    1. 选中 Checkbox 后 开始 按钮变为 Actived 状态，反选 Checkbox 见 MX-1-1. 登录前页面 - 未勾选；点击 用户协议 进入 MX-2-1. 用户协议
    2. 开始 按钮变为 Actived 状态，点击 开始 按钮进入 MX-3-1. 快速登录 - 未输入

    MX-2-1. 用户协议¶

    1. 点击 返回 按钮返回至前一个界面：MX-1-1. 登录前页面 - 未勾选 或 MX-1-2. 登录前页面 - 已勾选
    2. 文本信息
      • 若由于网络问题导致加载失败的话，见 MX-2-2. 用户协议 - 加载失败

    MX-2-2. 用户协议 - 加载失败¶

    1. 点击 返回 按钮返回至前一个界面：MX-1-1. 登录前页面 - 未勾选 或 MX-1-2. 登录前页面 - 已勾选
    2. 点击 重新加载 按钮进入 MX-2-3. 用户协议 - 重新加载

    MX-2-3. 用户协议 - 重新加载¶

    1. 尝试重新加载
      • 重新加载成功，进入 MX-2-1. 用户协议
      • 重新加载失败或 8 秒超时，返回 MX-2-2. 用户协议 - 加载失败

    MX-3-1. 快速登录 - 未输入¶

    1. 点击 取消 按钮返回至 MX-1-2. 登录前页面 - 已勾选
    2. 光标自动聚焦在 手机号码 一栏，并弹出 数字键盘
    3. 获取验证码 按钮默认 Disabled 状态，在 手机号码 输入完整后才变为 Actived 状态
    4. 点击可以输入验证码，见 MX-3-4. 快速登录 - Focus 验证码
    5. 登录 按钮默认 Disabled 状态，在 手机号码 及 验证码 两栏填写完毕后，才变为 Actived 状态

    MX-3-2. 快速登录 - 输入手机号完¶

    1. 点击 取消 按钮返回至 MX-1-2. 登录前页面 - 已勾选
    2. 输入手机号码时，有清除按钮
    3. 在 手机号码 输入完整后才变为 Actived 状态，点击获取验证码，见 MX-3-3. 快速登录 - 获取验证码，获取验证码 按钮在倒计时 60 秒结束后可重新点击
      • 若短信平台返回 获取验证码超出每天次数 的错误信息，则进入 MX-3-3-1. 快速登录 - 获取验证码超出每天限制次数
      • 若短信平台返回 该手机号已被拉入黑名单 的错误信息，则进入 MX-3-3-2. 快速登录 - 手机号被拉入黑名单
    4. 点击可以输入验证码，见 MX-3-4. 快速登录 - Focus 验证码
    5. 登录 按钮默认 Disabled 状态，在 手机号码 及 验证码 两栏填写完毕后，才变为 Actived 状态

    MX-3-3. 快速登录 - 获取验证码¶

    Success

    此提示为通知类 Toast 提示

    1. 提示停留 3 秒之后收起

    MX-3-3-1. 快速登录 - 获取验证码超出每天限制次数¶

    Failure

    此提示为警告 / 错误类 Toast 提示

    1. 点击 知道了 按钮关闭对话框，返回至 MX-3-2. 快速登录 - 输入手机号完

    MX-3-3-2. 快速登录 - 手机号被拉入黑名单¶

    Failure

    此提示为警告 / 错误类 Toast 提示

    1. 点击 知道了 按钮关闭对话框，返回至 MX-3-2. 快速登录 - 输入手机号完

    MX-3-4. 快速登录 - Focus 验证码¶

    1. 点击 取消 按钮返回至 MX-1-2. 登录前页面 - 已勾选
    2. 点击进行输入手机号码，见 MX-3-2. 快速登录 - 输入手机号完
    3. 在 手机号码 输入完整后才变为 Actived 状态，点击获取验证码，见 MX-3-3. 快速登录 - 获取验证码，获取验证码 按钮在倒计时 60 秒结束后可重新点击
      • 若短信平台返回 获取验证码超出每天次数 的错误信息，则进入 MX-3-3-1. 快速登录 - 获取验证码超出每天限制次数
      • 若短信平台返回 该手机号已被拉入黑名单 的错误信息，则进入 MX-3-3-2. 快速登录 - 手机号被拉入黑名单
    4. 光标聚焦在验证码一栏，最多可输入 6 位数字，输入完见 MX-3-4-1. 快速登录 - 填写验证码完
    5. 登录 按钮默认 Disabled 状态，在 手机号码 及 验证码 两栏填写完毕后，才变为 Actived 状态

    MX-3-4-1. 快速登录 - 填写验证码完¶

    1. 点击 取消 按钮返回至 MX-1-2. 登录前页面 - 已勾选
    2. 点击进行输入手机号码，见 MX-3-2. 快速登录 - 输入手机号完
    3. 在 手机号码 输入完整后才变为 Actived 状态，点击获取验证码，见 MX-3-3. 快速登录 - 获取验证码，获取验证码 按钮在倒计时 60 秒结束后可重新点击
      • 若短信平台返回 获取验证码超出每天次数 的错误信息，则进入 MX-3-3-1. 快速登录 - 获取验证码超出每天限制次数
      • 若短信平台返回 该手机号已被拉入黑名单 的错误信息，则进入 MX-3-3-2. 快速登录 - 手机号被拉入黑名单
    4. 输入验证码的时候，有清除按钮
    5. 在输入完 6 位验证码后，登录 按钮变为 Actived 状态。点击 登录 按钮进入 MX-3-5. 快速登录 - 验证中

    MX-3-5. 快速登录 - 验证中¶

    1. 进行登录验证
      • 若登录成功，进入 M1. 内容
      • 若检测到验证码有误，进入 MX-3-5-1. 快速登录 - 验证码错误
      • 若手机此时没有网络连接，进入 MX-3-5-2. 快速登录 - 手机没有网络连接

    MX-3-5-1. 快速登录 - 验证码错误¶

    Failure

    此提示为警告 / 错误类 Toast 提示

    1. 提示停留 3 秒之后收起，返回至 MX-3-4-1. 快速登录 - 填写验证码完

    MX-3-5-2. 快速登录 - 手机没有网络连接¶

    1. 点击 知道了 按钮关闭对话框，返回至 MX-3-4-1. 快速登录 - 填写验证码完
