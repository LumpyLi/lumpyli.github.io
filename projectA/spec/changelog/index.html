    Changelog¶

    All notable changes to this project will be documented in this file.

    The format is based on Keep a Changelog.

    [0.2.1] -2018-03-26¶

    Fixed¶

    • 修正 语聊 中「说话时间超长」状态下按钮状态的错误

    [0.2.1] - 2018-03-23¶

    Added¶

    • 在 妈妈讲故事 的配乐中，添加了能控制 人声 及 背景音乐 的控制面板

    Changed¶

    • Artboard 命名序号更改

    Fixed¶

    • 妈妈讲故事 中的 删除 按钮漏翻译问题

    [0.2.0] - 2018-03-22¶

    Added¶

    • 睡眠模式 增加小夜灯开关
    • 妈妈讲故事 增加 自由录制模式
    • 妈妈讲故事 增加 5 分钟的录音时间限制
    • 增加 我的收藏 播放时的交互
    • 语聊 中新增 连接中、收取中、加载失败 三种状态
    • 语聊 增加 亲子对话 未读消息的 Badge
    • 亲子对话 中增加 说话时间太短、接近 60 秒时倒计时、说话时长超长 三种状态

    Changed¶

    • 从 内容 中的分类进入由 二级分类 修改为 专辑列表
    • 修改 妈妈讲故事 中模版录音的交互
    • 修改 妈妈讲故事 中选择音乐的交互
    • 修改 妈妈讲故事 中 已录制 专辑下的 Table View 样式
    • 修改 我的上传 中专辑下的 Table View 样式

    Removed¶

    • 去除 内容 中的二级分类
    • 去除 妈妈讲故事 中的 试听、重新录制 两个功能
    • 去除 语聊 中没网络加载不出对话列表的交互

    [0.1.2] - 2018-03-21¶

    Changed¶

    • 分类 国学 修改成 国学经典

    [0.1.2] - 2018-03-16¶

    Added¶

    • 首页 增加 我的上传 列表
    • 我的 增加 我的上传 入口及扫码登录的交互流程

    Changed¶

    • 微调 首页 音频列表的间距

    [0.1.1] - 2018-03-15¶

    Added¶

    • 增加 专辑 的维度
    • 技能 中增加进入 Banner 文章的样式

    Changed¶

    • 内容 的分类由 5 个减少至 4 个
    • 内容 页面的 乐迪电台 更为突出，将 睡前故事 及 妈妈讲故事 平行处理
    • 内容 中的推荐改为 专辑
    • 调整 妈妈讲故事 的故事列表交互，类似米兔
    • 调整 技能 首页的页面样式及排布

    [0.1.1] - 2018-03-12¶

    Changed¶

    • 登录 页面中的 绑定 按钮文案改为 登录

    [0.1.0] - 2018-03-08¶

    Added¶

    • 增加 我的 模块的交互

    [0.1.0] - 2018-03-07¶

    Added¶

    • 语聊 模块交互
    • 技能 模块交互

    Fixed¶

    • 修复配网页面键盘文字丢失的问题

    [0.1.0] - 2018-03-06¶

    Added¶

    • 妈妈讲故事 模块的交互
    • 收藏列表 模块的交互

    [0.1.0] - 2018-03-05¶

    Added¶

    • 内容 大部分页面

    Changed¶

    • 登录 / 注册 系统改用 短信 + 验证码 进行登录及注册

    Removed¶

    • 去除淘宝百川登录系统

    [0.1.0] - 2018-03-02¶

    Added¶

    • 登录 交互
    • 配网 交互
